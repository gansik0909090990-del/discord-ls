Данный бот работает совместно с сайтом предназначенным для обучения новичков, которые вступают в органы прокуратуры на проекте RMRP. Задача данного бота заключается в:
Уведомлять пользователей о том, что их заявка на регистрацию одобрена администратором
Уведомлять об разблокировке нового экзамена для дальнейшего прохождения
Уведомлять о выдаче выговора/предупреждения
Уведомлять о провале экзамена в связи с попыткой списать
Уведомлять о том, что пользователю дан второй шанс на прохождение экзамена.
